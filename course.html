INTRO TO HTML,CSS & JAVASCRIPT



  1. HTML stands for Hypertext Markup Language.
  2. HTML is the standard document format for web pages which is supported by all browsers.
  3. HTML consists of a series of elements, which you use to enclose, wrap, or mark up different parts of content to make it appear or act in a certain way.
  1. CSS stands for Cascading Style Sheets
  2. CSS saves a lot of work. It can control the layout of multiple web pages all at once
  3. CSS describes how HTML elements are to be displayed on screen, paper, or in other media
    1. Inline CSS
    2. Internal/ Embedded CSS
    3. External CSS
  1. JavaScript (JS) is the most popular lightweight, interpreted compiled programming language.
  2. It can be used for both Client-side as well as Server-side developments.
  3. JavaScript also known as a scripting language for web pages.
